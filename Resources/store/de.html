Dieses Plugin erlaubt tiefere Einblicke in die API Abläufe. The festgehaltenen REST API Aufrufe werden in shopware-root/var/log/rest_api_*.log Dateien abgelegt.

Folgende Informationen werden festgehalten:
  • Methode
  • URL
  • Benutzer
  • Übertragene Daten

Dieses Plugin wird von @FriendsOfShopware entwickelt. Maintainer dieses Plugins ist: Joshua Behrens
